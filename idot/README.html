ιDOT: A DOT Calculus with Safe Object Initialization

Our paper presents ιDOT calculus, which extends the κDOT calculus of Kabir and Lhoták with a type and effect system object to ensure null-safety and makes fields non-lazy. κDOT is itself an extension of the WadlerFest DOT calculus of Amin el al. (2016). This repository contains a formalization of the type-safety proof of the calculus and various extensions of the calculus in Coq.

Documentation

We provide three different versions of the calculus.

  • The base ιDOT calculus, as described in the main sections of our paper.
  • Extension of ιDOT with Free Literals, as described in section 6.1
  • Extension of ιDOT with Local Initialization, as described in section 6.2
You may explore the documentation for the various calculi by following the boldface links in the above list.
